File Links:
===========
<a> tag
<a href = "file-location"> Link Text </a>

<!DOCTYPE html>
<html>
    <head>
        <title> File Links </title>
    </head>
    <body>
        <h1> Welcome To My Website </h1>
        <a href = "G:\UI_4pm\Day_18\Navigation Bar.txt" target="_blank" name = "myFile"> Get The File </a>
    </body>
</html>

<iframe>
========
==> used to open an website in another website.
Syntax:
	<iframe src = "address" width height>

Attribute:
==========
frameborder = "0" ==> to off the border for iframes.

<!DOCTYPE html>
<html>
    <head>
        <title> IFrames in HTML </title>
    </head>
    <body>
        <h1> Welcome to Ashok IT </h1>
